CHIP-8 Disassembler

This is a CHIP-8 disassembler written in JavaScript.

It is supposed to work with my CHIP-8 emulator.

Usage

  1. Initialize my CHIP-8 emulator instance
chip8.initialize({
    ctx: document.querySelector('canvas').getContext('2d')
});
  1. Load ROM into the memory
chip8.loadROM('roms/Puzzle.ch8', onROMLoaded);
  1. Attach disassembler to emulator instance and decode
function onROMLoaded() {
    var dasm = new Chip8Dasm(chip8);
    var result = dasm.disassemble(0x0200);
    // ...or decode single instruction:
    // dasm.decode(0x2666)
}
